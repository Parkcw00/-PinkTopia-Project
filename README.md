# PinkTopia (5조)

닌텐도는 혁신적인 게임 경험을 선사하는 회사로, 오랜 시간 동안 전 세계 팬들에게 사랑받아왔습니다. **커비게임**은 귀여운 핑크색 캐릭터 커비와 그의 독특한 능력 흡수 플레이로 창의적이고 즐거운 모험을 제공하며, 닌텐도의 매력을 잘 보여줍니다. 무엇보다, 포켓몬Go!는 현실과 가상 세계를 연결해 플레이어들이 밖으로 나와 포켓몬을 잡고 친구들과 소통하게 하며, 건강한 재미를 선물합니다. 이들은 모두 닌텐도의 독창성과 끝없는 즐거움을 상징하는 보물 같은 존재입니다!

## 🎮 게임 시나리오

어느 날, 먼 은하계에서 전해지던 전설 속 생명체들 — **핑크 빛을 띤 '핑크몽'**들이 예기치 못한 위기를 피해 지구에 발을 들여놓았습니다. 그들의 존재는 신비로움 그 자체였으나, 인간과는 전혀 다른 언어와 문화로 인해 소통의 벽이 높았습니다. 그럼에도 불구하고, 핑크몽들의 순수한 마음과 독특한 매력은 한국의 한 시민 단체에게 깊은 감동을 주었습니다.

이 단체는 바로 **‘핑크 군단’**. 핑크몽들이 새로운 환경에 적응하고 안전하게 지낼 수 있도록 정성껏 음식을 제공하며 돌보는 일을 자처합니다. 핑크 군단은 단순한 도움을 넘어, 서로 다른 두 존재가 진정한 우정을 나눌 수 있다는 믿음 아래, 인간과 핑크몽 사이의 다리를 놓고자 합니다.

이제, 당신은 핑크 군단의 일원으로서 핑크몽들에게 따뜻한 관심과 사랑을 전하며, 그들과 함께 조화로운 미래를 만들어 나가는 모험에 동참하게 됩니다.

## 🎯 서비스 기획 의도

본 서비스는 운동이 부족한 **청소년과 청년들**에게 **집 밖의 다양한 즐거움**을 제공하여, 자연스럽게 **외출과 활동을 유도**하는 것을 목표로 합니다. 구체적인 기획 의도는 다음과 같습니다:

1. **건강 증진 및 외출 유도**  
   - 사용자가 집 밖으로 나와 다양한 목적지를 탐색하며 **일상 속 운동**을 실천하도록 돕습니다.  
   - 출발-도착 마커 기능으로 길 찾기 어려움 없이 안전한 이동을 지원합니다.

2. **게임 요소를 통한 흥미 유발**  
   - 퀘스트와 귀여운 핑크몽 수집 요소로 단순 산책을 **목표 있는 모험**으로 변모시킵니다.  
   - 문제 해결 능력과 도전 정신을 자연스럽게 키웁니다.

3. **사회적 소통 및 커뮤니티 형성**  
   - 채팅 기능을 통해 경험을 공유하며 **소통 능력 향상**과 커뮤니티 형성에 기여합니다.  
   - 함께 목표를 달성하며 **사회적 유대감**을 강화합니다.

4. **지역사회 활성화**  
   - 지역 명소와 문화를 경험할 기회를 제공합니다.  
   - 새로운 장소에 대한 관심과 방문을 촉진합니다.

> **핵심 목표**: 건강한 라이프스타일, 즐거운 소통, 지역사회 활성화를 동시에 추구하며, 사용자에게 의미 있는 경험을 선사합니다.

## 🛠️ 기술 스택 및 기술적 의사결정

### **Backend**
| 기술 스택    | 선택 이유                                                                 |
|--------------|--------------------------------------------------------------------------|
| **NestJS**   | 모듈화된 아키텍처와 타입스크립트 기반으로 유지보수성과 확장성이 뛰어남    |
| **Socket.IO**| 실시간 채팅과 위치 정보 제공을 위한 빠르고 효율적인 통신 지원             |
| **Geolocation**| 사용자 위치 기반 실시간 서비스 제공                                      |
| **TypeORM**  | 타입 명시로 코드 오류 감소                                               |

### **Frontend**
| 기술 스택            | 선택 이유                                                                 |
|----------------------|--------------------------------------------------------------------------|
| **HTML/CSS/JS**      | 직관적인 UI, 모바일 디자인 조정, 실시간 데이터 업데이트 및 동적 기능 추가 |

### **Database**
| 기술 스택   | 선택 이유                                                                 |
|-------------|--------------------------------------------------------------------------|
| **AWS S3**  | 이미지 데이터의 안전한 저장과 백업 관리                                  |
| **MySQL**   | 간단한 설정, NestJS와 쉬운 연동, 무료 오픈소스, 데이터 무결성 보장       |
| **Valkey**  | 고속 데이터 처리로 실시간 기능 속도 향상 및 DB 부하 감소                 |

### **DevOps / Infra**
| 기술 스택   | 선택 이유                                                                 |
|-------------|--------------------------------------------------------------------------|
| **Docker**  | 배포 효율성, 환경 일관성 유지, 수평 확장 가능                           |
| **Sentry**  | 실시간 에러 모니터링 및 빠른 대응, Slack 연동으로 장애 대응 시간 단축    |
